Toss a coin 100,000 times. We will first check our random number generator – is it really 50% chance heads/tails?

set.seed(12345)

nFlips <- 100000

Flips <-sample(0:1, nFlips, repl=T)

Trajectory <- cumsum(Flips)

nTraj <- Trajectory/(1:nFlips)
par(mfrow=c(1,1))
plot(nTraj, ylim=c(.3,.7), type="l", col="blue")
lines(c(0,nFlips),c(.5,.5))

INTERPRETATION

As we move along the X axis (flip more and more times), the percentage of total flips that is heads (or tails) becomes very close to .5, as we expected. This indicates that the coin is a fair coin. Over the short term, there are more 1's than 0's, but iafter about 20000 observations of the coin flip the number of 1's becomes very close to the number of 0's when expressed as a percentage of total flips.

The following lines will generate a trajectory that goes up one step if the current outcome is Tails and down one step if the outcome is Heads. The total number of steps is now 1,000,000.

Another check of intuition. What do you expect the probabilities of the following events be:

“In 500 coin tosses the difference between the number of Heads and the number of Tails is less than 5 in absolute value” I expect the probability of this is .2

“In 500 coin tosses the difference between the number of Heads and the number of Tails is greater than 25 in absolute value” I expect the probability of this is .05

Find out by transforming the 1,000,000 flips into 2000x500 flip matrix 

Flips.matrix <- matrix(data=Flips.plusminus1, nrow=2000, ncol=500)
Trajectory.matrix <- t(apply(Flips.matrix, MARGIN=c(1), cumsum))

# take last row -- this is the last sums
final.difference <- Trajectory.matrix[,500]

probability.less.than.5 <- mean(abs(final.difference) < 5 )
probability.greater.than.25 <- mean(abs(final.difference) > 25)

The probability of the difference between the number of heads and the number of tails being less than 5 was found to be probability.less.than.5.

The probability of the difference between the number of heads and the number of tails being greater than 25 was found to be probability.greater.than.25.

INTERPRETATION

Even though we know the coin is fair, we found that there is a sizeable chance (~25%) that you will find over 25 more heads than tails in 500 Flips. This was more than my intuition.

The following function determines the time intervals between crossings:

TimeIntervalsBetweenCrossings <- function(CrossingTimes){
  if(length(CrossingTimes) == 0) { 
    return(0)
  }else{
    xtimes.sort <- sort(CrossingTimes[CrossingTimes>0])
    return(sum(diff(xtimes.sort) -1))
  }
}

Apply that function across the 2000 runs and plot the histogram

hist((sapply(ZeroCrossingTimes2000Samples, TimeIntervalsBetweenCrossings)), main="Histogram", xlab="Sum of zero crosses")

The law that describes this behavior is the Arc-Sine Law. We see this behavior because the further the process gets from the origin, the longer it is expected to take to return to the origin.
